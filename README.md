# 🎮 Horror SMP

Welcome to the official **From The Flingue** repository!

This repository contains the files used for the Minecraft server, including mods, configurations, and other project resources.

📦 Contents

* 📁 Mods
* ⚙️ Configurations
* 📄 Server files

📥 Installation

1. Download the repository content.
2. Place the mods into the `mods` folder.
3. Launch your Minecraft server or client with the correct version.
